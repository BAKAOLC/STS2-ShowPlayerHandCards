# Show Player Hand Cards

A mod for Slay the Spire 2 that displays teammate hand cards next to the multiplayer player list in real-time.

## Features

- Displays scaled-down full card images next to each teammate's player state bar
- Cards update in real-time as teammates draw and play cards
- Hover over any mini card to view the full card details via hover tip
- Uses spacer-in-layout + sync-position pattern for correct UI layering
- Only shows for non-local players during combat
